   Projects

  • Subscription Manager

    This is my Senior Year Capstone Project. This was done for a client, Moneytree. Part of Accutech. The project is written using VueJS for the frontend framework, with an ASP.Net Web API for the backend. I signed an NDA before working on this project, so I cannot link any code. But, I can describe it as best as possible.

    The project consists of three parts:
    • A Bulk Signup Software
    • A Subscription Manager Dashboard Software
    • A User Onboarding Software

    The Bulk Signup Software is a solution to companies having to submit a form for every individual they want to signup for Moneytree products. It allows the user to dynamically add and fillout as many signups as they want (to the limit of 20). This allows the user to fill out payment information only one time. This signup information is submitted to a thirdparty service, Maxio, which handles their product subscriptions.

    The Susbcription Manager Dashboard is a software which exists under each of Moneytree's different prodcts, with a dashboard specifically written for each product. The dashboard pulls relevant information about the signed in customer, as well as relevant information for the product they are signed into. The user can view all information, update their billing/personal information, and view/pay invoices.

    The User Onboarding Software is a software which asks the user multiple questions when they are signing up, granting Moneytree more information to taylor their experience too.

    All of the information gathered/pulled in this project was gotten from or store in Maxio.

    I took most of the lead on this project, by planning meetings with the team/client each week, and also encouraging communication between team members. I also planned out each iteration.

  • The Perfect Bet

    This was a program completely focused on sports betting. The project started off just focusing on NFL games, and would update week to week with current matchups. A free sports betting API was used to get the matchups, as well as the odds for each game. The user was able to select a matchup from a list of games, and enter a bet amount on one team. Using the odds, and the amount placed, the program would calculate their potential winnings.

    In later iterations, NBA and NHL games were added. A feature was also added that allowed the user to select certain important statistics from a specific sport. The selected statistics for the selected team would display which allowed the user to make a more informed bet. There was a heavy emphasis placed on unit testing in this project using JUnit, and JavaFX was used to create the GUI. This project was a 10+ week project during my second year.
